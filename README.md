# Tela de Login utilizando React Native (Expo)

## Como utilizar o projeto
Clone Repositório
```sh
git clone git@github.com:allanvigiani/react-native-login.git
```
```sh
cd react-native-login/LoginApp
```

Instale as dependências do projeto
```sh
npm install
```

Após a conclusão da instalação das dependências, rode o seguinte comanda para inicializar o projeto
```sh
npx expo start
```

Após isso, aparecerá um QrCode. Escaneie com seu celular no aplicativo Expo Go ou aperte 'w' para abrir em seu navegador.

Acesse o projeto
[http://localhost:8001](http://localhost:8081)
